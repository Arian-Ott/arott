# Arott OS

Information: This is a non-stable OS for educational purposes.


## Getting started

If you want to run ArottOS (preferably within VM), you have to build the iso using the provided `Makefile`.
Once built, you can use the iso to boot into the machine.

Following System requirements were tested inside a Debian 13 VM in Virtual Box.

**System requirements:**
- 1 CPU Core (tested with Intel)
- Minimum 5 MiB RAM, recommended 16 MiB or 32 MiB
- No Harddrive needed
- No Ethernet connection needed
